# Week 3 Side Quest - Portal Brawl

## Project Title
Portal Brawl (Simple Fighting Game)

---

## Setup and Interaction Instructions

### How to Run the Game
1. Open the project folder in VS Code
2. Run using Live Server
3. Open the game in Google Chrome

---

### Game States
- Start Screen
- Fight Screen
- Win Screen

Press **ENTER** to start and restart the game.

---

## How to Play

### Player 1 (Blue)
- Move Left: A
- Move Right: D
- Attack: F

### Player 2 (Orange)
- Move Left: Left Arrow
- Move Right: Right Arrow
- Attack: K

---

## Game Features

- Simple 2-player fighting system
- Health system (3 HP each)
- Attack hit detection based on distance
- Start, fight, and win screens
- Portal-inspired theme (blue vs orange test subjects)
- Background music during gameplay
- Punch sound on attack
- Victory sound on win
